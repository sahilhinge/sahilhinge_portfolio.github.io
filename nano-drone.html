Nano DIY Drone

Discover the build process, features, and gallery of the Nano DIY Drone project.

Prototype

The initial prototype was built using an ESP32-WROOM module integrated with an onboard IMU, enabling Wi-Fi-based control and basic flight stabilization for the Nano DIY drone.

Alt Hold feature

This version featured altitude hold using a ToF sensor and included a custom-designed PCB developed from schematic to layout, enhancing integration and performance

Iteration

Following RCA analysis and multiple design iterations, a custom PCB was developed from schematic to layout, integrating a ToF sensor for reliable altitude hold.
